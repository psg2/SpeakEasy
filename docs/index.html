OpenVoicy

Speech-to-Text for macOS

A native macOS app with global hotkey support. Transcribe your voice anywhere with a single keyboard shortcut. Choose between cloud-powered OpenAI Whisper or privacy-focused local transcription.

Download View on GitHub

Features

Global Hotkey

Trigger recording from anywhere with a keyboard shortcut. Default: Option + Space

Dual Providers

Choose between OpenAI's cloud API for accuracy or on-device WhisperKit for privacy.

Auto-Type

Transcribed text is automatically typed into the focused application. No copy-paste needed.

Floating Overlay

Animated waveform UI shows recording status in real-time without interrupting your workflow.

History

Searchable transcription history with date grouping and audio playback capability.

Model Management

Download and manage local Whisper models with progress tracking. Pick the right size for your needs.

How It Works

1

Press the Hotkey

Hit Option + Space from any application to start recording.

2

Speak

Say what you want to type. The floating overlay shows you're recording.

3

Release

Release the shortcut and your words are transcribed and typed automatically.

Requirements

  • macOS 14.0 (Sonoma) or later
  • Apple Silicon recommended for local transcription
  • Microphone access permission
  • Accessibility permission (for auto-typing)
